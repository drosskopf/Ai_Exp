# Initial Setup

## Description
This will set up the initial project

## Requirements

- Use .Net Aspire 
- Create the following
    - .Web.Api: Dotnet 10 webapi project
    - .Web: NextJs application
    - .Web.Api.Integration.Test: Test project for the Api
    - .Web.Api.Unit.Test: Unit Test Project for the Api
    - .Web.Test: Playwright test, hooked into the aspire framework.  
- Both apps should be registered in the AppHost
